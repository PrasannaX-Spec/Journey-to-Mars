# 🚀 Journey to Mars

A cinematic, scroll-driven space experience from Earth to Mars with immersive storytelling and interactive exploration.

🔗 Live Demo: (add after deployment)

## 🌌 Overview
A cinematic scroll-based web experience that simulates the journey from Earth to Mars. Designed to meet all the mandatory requirements of the creative challenge, the project features a cohesive narrative that combines advanced 3D integrations, premium interaction design, and data-driven storytelling.

## 🛠 Tech Stack
- **Frontend:** HTML, CSS, JavaScript (Vanilla)
- **Animation Libraries:** GSAP, ScrollTrigger, Native CSS Animations
- **3D Integration:** Spline, Three.js (via Runtime)
- **Smooth Scroll:** Lenis

---

## 🎯 Mandatory Requirements Handled

### 📖 1. Story Structure (5 Sections)
The experience forms a cohesive interplanetary narrative structured into **5 distinct phases**:
1. **Phase 1: Initiating (Hero):** Pre-flight logs, telemetry introduction, and Starliner-9 mission details.
2. **Phase 2: Launch (Atmospheric Breach):** High-velocity exit from Earth's atmosphere, complete with dynamic G-Force tracking.
3. **Phase 3: Transit (Deep Space):** A reflective mid-journey phase highlighting time elapsed and the vast distance.
4. **Phase 4: Descent (Mars Entry):** Intense re-entry simulating heat-shield telemetry and radar altimeter tracking.
5. **Phase 5: Touchdown (Mission Success):** Arrival at Jezero Crater, presenting surface conditions and the final CTA.

### 🖱️ 2. Scroll-Based Interaction
- **Scroll-Triggered Timeline:** The entire narrative unspools dynamically via GSAP ScrollTrigger, advancing scenes precisely based on vertical scroll scrub.
- **Deep Space Parallax:** Procedural starfields, nebula layers, and atmospheric debris move at varying fractional speeds relative to the user's scroll.

### 🎮 3. Interactive Elements
- **Interactive Telemetry Tooltips:** Hovering over critical HUD readouts (like Altitude or Velocity) reveals technical tooltips with animated SVG sparklines.
- **Magnetic Call-to-Action:** The final "Explore the Surface" CTA button features dynamic magnetic hover physics that follow the user's cursor.
- **HUD Systems Toggles:** Clickable UI components in the Heads-Up Display (Comms, Power, Life Support) that toggle active states and alert styles.

### 🎬 4. Animations
- **Glitch Text Reveal:** Mission logs display an algorithmic character-scrambling "glitch" effect upon scrolling into view.
- **3D Spline Transition:** A full 3D Spline Earth/Mars scene dynamically loaded and integrated into the viewport timeline.
- **Synchronized Loading SVGs:** An immersive, multi-step pre-flight loading sequence, plus infinite rotating HUD scanner SVGs.
